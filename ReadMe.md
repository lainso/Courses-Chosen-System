简介：
	本项目是一个学生考勤信息管理系统，用户登录识别自动打开对应身份页面
	本项目采用MVC模式，基于 java swing + mysql 进行开发
	管理员端提供：
		1.学生信息 增删改查
		2.教师信息 增删改查
		3.课程信息 增删改查
	教师端提供：
		1.考勤记录信息 增改查 统计
	学生端提供：
		1.考勤记录信息 查

使用方法：
	1.打开 .\src\com\artisan\util\DbUtil.java
		修改 11-14 行中的数据库连接配置
	2.打开本地数据库，导入 .\student.sql 文件以加载测试数据
	3.运行 .\src\com\artisan\view\Login.java 以开始使用
